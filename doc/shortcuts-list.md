# Shortcuts in ZXLive

| Action                   | Shortcut           | Applicable Panels|
|--------------------------|--------------------|------------------|
| New Graph Tab            | Ctrl+N             | All              |
| New Window               | Ctrl+Shift+N       | All              |
| Toggle Sound Effects     | Ctrl+B             | All              |
| Copy TikZ to Clipboard   | Ctrl+Shift+C       | All              |
| Deselect All             | Ctrl+D             | All              |
| Select                   | S                  | All              |
| Paste TikZ from Clipboard| Ctrl+Shift+V       | Base             |
| Vertex                   | V                  | Base             |
| Edge                     | E                  | Base             |
| Snap                     | F                  | Base             |
| Toggle Sparkles          | Ctrl+Shift+Alt+S   | Proof            |
| Magic Wand               | W                  | Proof            |
| Dummy Vertex             | V                  | Proof            |
| Dummy Edge               | E                  | Proof            |
